# CXPOS

Modern Point of Sale Solution for Retail & Business Management.

CXPOS is a fast, reliable, and easy-to-use POS application designed to simplify daily business operations. Built with modern technology and optimized for performance, CXPOS helps businesses manage sales, billing, inventory, customers, and reporting efficiently.

Whether you run a retail shop, supermarket, restaurant, pharmacy, or wholesale business, CXPOS provides the tools needed to improve productivity and streamline operations.

## Features

- Fast billing and invoicing
- Inventory and stock management
- Customer management
- Sales and profit reports
- Barcode scanner support
- Thermal printer support
- Multi-user access
- Offline-ready desktop application
- Secure and stable performance
- Easy update and maintenance process

## Technology

CXPOS is built using:

- Vue.js
- Electron
- Node.js
- SQLite / MySQL support

## Updates

Application updates are released periodically to improve stability, performance, and features. Updates are manually distributed by the support team when required.
